# All in Copilot

Use your own AI models directly in VS Code Copilot Chat.

English | [简体中文](https://github.com/goldenfishs/all-in-copilot/blob/main/README.zh-CN.md)

</div>

## Supported Models

OpenAI, Claude, DeepSeek, Gemini, xAI, Kimi, MiniMax, GLM, and any OpenAI-compatible API.

## Install

Search **All in Copilot** in VS Code Extensions and install it.

Requirements: VS Code 1.116+ and GitHub Copilot Chat.

## How to Use

1. Open **All in Copilot** in the Activity Bar.
2. Choose a provider preset or enter your own Base URL.
3. Paste your API key.
4. Click **Fetch Models** to discover available models.
5. Select a model in Copilot Chat.

## Features

- Visual model management (add, edit, delete)
- Auto-detect Base URL format
- Per-provider API key storage (not in settings.json)
- Reasoning effort configuration

## Commands

- `All in Copilot: Open Model Manager` — Open the sidebar
- `All in Copilot: Set Provider API Key` — Configure API key
- `All in Copilot: Open Settings` — Extension settings
