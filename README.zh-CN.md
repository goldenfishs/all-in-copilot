# All in Copilot

在 VS Code Copilot Chat 里直接使用你自己的 AI 模型。

[English](https://github.com/goldenfishs/all-in-copilot#readme) | 简体中文

</div>

## 支持的模型

OpenAI、Claude、DeepSeek、Gemini、xAI、Kimi、MiniMax、GLM，以及任何 OpenAI 兼容接口。

## 安装

在 VS Code 扩展商店搜索 **All in Copilot** 并安装。

要求：VS Code 1.116+ 和 GitHub Copilot Chat。

## 使用方法

1. 打开左侧 Activity Bar 的 **All in Copilot**。
2. 选择一个服务商预设，或填入你自己的 Base URL。
3. 输入 API Key。
4. 点击 **获取模型列表** 发现可用模型。
5. 在 Copilot Chat 中选择模型。

## 功能

- 可视化管理模型（添加、编辑、删除）
- 自动识别 Base URL 格式
- 按服务商保存 API Key（不在 settings.json 中）
- 推理强度配置

## 命令

- `All in Copilot: 打开模型管理器` — 打开侧边栏
- `All in Copilot: 设置服务商 API Key` — 配置 API 密钥
- `All in Copilot: 打开设置` — 扩展设置
